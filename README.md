
# react-native-code-push-test
React Native Code Push Test 앱 

## Prerequisites

- [Node.js](https://nodejs.org/)
    - Version 20 이상
- [Ruby]()
    - Version 3.0.5


## Getting Started

### Installation

```bash
yarn install
bundle install

# ios
yarn pod-install
or
yarn pod-install-old
```
